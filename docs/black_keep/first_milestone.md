# First Milestone

The first Black Keep milestone should prove the final game identity while staying buildable on top of the current SpriteVania vertical slice.

## Route

```text
Title Screen
-> Character Creation: choose Ronin / Arc-Gunner / Iron Knight
-> Modern City Outskirts
-> Rural Swamp Road
-> Swamp Tutorial
-> Castle Gate
-> Damaged Shrine
-> Recruit The Black Witch of Ash
-> Full active-party UI unlocks
-> Momentum / tag-swap tutorial
-> Samurai Castle Wing
-> Stealth/rescue section
-> Rescue The Shadow
-> Player names The Shadow
-> Three-character active party online
-> Boss: Lord Masakiro, Warlord of the Black Keep
-> Masakiro is consumed by the oni after defeat
-> Unlock Rising Torii Seal
-> Enter Sakuramori Court
-> Meet High Priestess Harune
-> Moonpetal Passage hinted, not fully unlocked
```

## Active Party Target

By the end of the Samurai Castle Wing, the active party should contain:

1. Player-chosen starter.
2. The Black Witch of Ash.
3. The Shadow.

## Systems Proven

The first milestone should validate:

- Three-character active party.
- One visible character at a time.
- Mid-fight swapping.
- Individual HP/resource bars.
- Individual Momentum rings.
- Tag-entry attacks.
- KO auto-switching.
- Benched active-character recovery.
- Hub party management.
- First major boss and seal reward.

## First Hub Arrival

Sakuramori Court is the first Cherry Blossom Court hub. It should be peaceful on first discovery and become visibly damaged after the World Break.

Leader: High Priestess Harune.

The Moonpetal Passage should be hinted here, but full fast travel/time-state functionality can arrive later.
