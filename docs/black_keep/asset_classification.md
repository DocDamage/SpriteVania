# Asset Classification

Once all asset folders are visible, classify every sprite and asset by gameplay use, visual compatibility, and fallback value.

## Primary Categories

- Playable.
- NPC.
- Enemy.
- Boss.
- Shopkeeper.
- Hub-only.
- Background/parallax.
- UI.
- VFX.
- Tile/terrain.
- Trap/hazard.

## Character Sorting Criteria

Character assets should be sorted by:

1. Animation completeness.
2. Scale compatibility.
3. Readable silhouette.
4. Combat usability.
5. Palette/style match.
6. Role fit.
7. NPC/enemy/boss fallback value.

## Playable Candidate Notes

Playable characters need enough animation support to sell movement, combat, traversal, hit reactions, and upgrades. Missing special-case animations can be filled later if the base silhouette, role, and animation quality are strong.

Monstrous sprites may be considered for both The Blood-Marked and The Yokai-Bound.
